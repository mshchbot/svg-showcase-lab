05 · Scroll Reveal

The Techniques Behind This Page

Every animation here uses vanilla web platform APIs — zero frameworks, zero libraries.

🎯

stroke-dasharray

The line-draw effect: set dash length to path length, animate offset from full to zero. Works on any stroked SVG path.

stroke-dasharray: 600;
stroke-dashoffset: 600 → 0
🌀

CSS @keyframes

GPU-composited transforms on SVG elements. Floating shapes, spinners, bouncing dots — all driven by keyframe animations.

@keyframes floatA {
  0%,100% { translate: 0 0 }
  50% { translate: 30px -40px }
}
👁️

IntersectionObserver

Sections animate into view as you scroll. The bar chart and info cards use this for reveal-on-scroll behavior.

new IntersectionObserver(…)
.observe(element)
